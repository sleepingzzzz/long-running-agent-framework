## 核心概念

### Workflows vs Agents

- **Workflows**: 通过预定义代码路径编排LLM和工具
- **Agents**: LLM动态指导自己的流程和工具使用

### 常见模式

1. **Prompt Chaining** - 提示链
2. **Routing** - 路由分类
3. **Parallelization** - 并行化
4. **Orchestrator-Workers** - 协调器-工作者
5. **Evaluator-Optimizer** - 评估器-优化器
